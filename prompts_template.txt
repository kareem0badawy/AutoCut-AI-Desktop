SCENE TYPES — pick the most fitting type for each moment:

TYPE A — CHARACTER SCENE
When the script mentions a person, profession, or human story.

Example: "exhausted weimar-era factory worker sitting on steps outside a shuttered berlin factory, head in hands, worn coat, empty lunch pail beside him"

TYPE B — OBJECT / SYMBOL SCENE
When the script mentions an object, currency, commodity, or concept.
Make the object the hero. Fill the frame with it. Use contrast and scale.
Example: "single gleaming gold bar resting on crumbling paper reichsmarks, close-up, dramatic lighting from above"

TYPE C — LANDSCAPE / ENVIRONMENT SCENE
When the script mentions a place, country, era, or large-scale event.
Show the world. Wide angle. Atmosphere over detail.
Example: "vast wheat fields stretching to the horizon outside berlin 1923, golden afternoon light, a lone farmhouse in the distance"

TYPE D — CONTRAST / COMPARISON SCENE
When the script compares two things, two groups, or two outcomes.
Split the composition. Show both sides. Let the image tell the contrast story.
Example: "split composition — left side: city banker surrounded by worthless paper notes in a cold berlin office / right side: confident farmer surrounded by grain sacks in warm sunlight"

TYPE E — ABSTRACT / METAPHOR SCENE
When the script talks about systems, patterns, history, or ideas.
Translate the idea into a visual metaphor. Think like a documentary director.
Example: "a crumbling stone wall with old banknotes plastered over cracks like wallpaper, peeling away to reveal solid gold bricks underneath"
