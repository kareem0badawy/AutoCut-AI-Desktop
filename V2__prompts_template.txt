SCRIPT-TO-IMAGE ALIGNMENT — MOST IMPORTANT RULE

This is the single most critical rule in this entire prompt.

THE RULE:
Every main_prompt must be a direct visual translation of what the script is saying at that exact moment.
The image must show WHAT THE SCRIPT IS TALKING ABOUT — not a generic historical scene.

HOW TO DO IT — FOLLOW THESE 4 STEPS FOR EVERY SCENE:

STEP 1 — FIND THE SCRIPT MOMENT
Read the script segment carefully.
Divide it mentally into {scenes_in_batch} moments — one moment per scene.
Each scene covers a specific sentence or idea in the script.

STEP 2 — EXTRACT THE CORE VISUAL
Ask: what is the single most iconic image for THIS moment in the script?
Not the general topic. Not the era. THIS specific sentence or idea.

Example:
Script says: "She rents a clean apartment in Brooklyn for $8 a month"
→ Core visual: a modest but tidy apartment interior, woman at ease
→ NOT: a generic historical cityscape

Script says: "On December 23rd, in Washington D.C., a bill was signed"
→ Core visual: a hand signing a document in a formal chamber, dramatic light
→ NOT: a random building exterior

Script says: "She puts her savings in the bank — because that's what careful people do"
→ Core visual: a woman placing a worn envelope into a bank teller's hands
→ NOT: a pile of coins or a generic vault

STEP 3 — BUILD THE main_prompt AROUND THAT VISUAL
The main_prompt must contain the specific person, object, or place from the script moment.
If the script mentions Margaret → show a woman matching her description.
If the script mentions a bank → show the act of banking, not a symbolic key.
If the script mentions wheat prices rising → show wheat, a scale, a farmer — not an abstract collapse.

STEP 4 — VERIFY BEFORE SUBMITTING
Ask yourself: if someone reads scene_description and then sees the image, will they say "yes, that's exactly it"?
If yes → submit.
If no → rewrite main_prompt to match the script moment.

❌ BAD — script says "Margaret saves $40 a year" → main_prompt shows "a symbolic key unlocking a vault"
✓ GOOD — script says "Margaret saves $40 a year" → main_prompt shows "a schoolteacher woman seated at a modest desk, carefully placing folded banknotes into a plain envelope, warm amber lamplight, quiet dignity"

SCENE TYPES — pick the most fitting type for each moment:

TYPE A — CHARACTER SCENE
When the script mentions a person, profession, or human story.
One character, one posture, one setting. No crowds.
FLUX example: "elderly schoolteacher seated at worn wooden desk, hands folded, looking at camera with quiet dignity, warm amber light from window"

TYPE B — OBJECT / SYMBOL SCENE
When the script mentions an object, currency, commodity, or abstract concept.
Object fills the frame. Negative space around it. Maximum one supporting prop.
FLUX example: "single gold bar resting on worn parchment surface, dramatic top-down lighting, deep shadow on one side"

TYPE C — LANDSCAPE / ENVIRONMENT SCENE
When the script mentions a place, country, era, or scale.
Wide angle. Atmosphere dominates. Maximum two elements.
FLUX example: "vast empty wheat field stretching to horizon, single farmhouse in far distance, golden afternoon light, overcast sky"

TYPE D — CONTRAST / COMPARISON SCENE
When the script compares two outcomes, two groups, or two worlds.
Split frame. Both sides readable. One warm, one cold.
FLUX example: "split composition — left half: cold grey empty bank vault interior / right half: warm sunlit grain storage barn"

TYPE E — ABSTRACT / METAPHOR SCENE
When the script discusses systems, patterns, cycles, or ideas.
One dominant metaphor. Visual idea over literal event.
FLUX example: "crumbling stone wall with blank paper sheets plastered over cracks, peeling away to reveal solid stone underneath, warm dramatic light"
